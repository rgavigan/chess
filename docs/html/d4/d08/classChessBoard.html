Determines if a potential spot would put a King in check.

Check if a square is safe from enemy attacks, particularly for the King.

Parameters
posThe potential position a King seeks to move to.
colourThe colour of the King trying to make the move.
Returns
true if the move would not result in check, false otherwise.

Check if the square is a valid move (i.e. is being attacked by) an enemy piece that is not a pawn. Pawns are handled separately

If the piece type is a pawn, ensure the column numbers are different (i.e. ensure it would be a diagonal capture) as pawns can't capture pieces directly in front of them.
