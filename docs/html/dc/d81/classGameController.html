Detailed Description

Author: Bradley McGlynn Description: Declaration of the GameController class. This class acts as a mediator between the game's logic and the game's state, managed by the GameState class, providing a unified interface for the UI to interact with. It is responsible for controlling the flow of the game, managing turns, and handling user interactions such as moves, undos, redos, saves, loads, and resignations. Date: September 28th, 2023
